==========================================================
  ALARM - монітор повітряної тривоги для м. Київ
==========================================================

Команда "alarm" у терміналі стежить за повітряною тривогою.
Щойно в Києві оголошують тривогу:
  - фон терміналу стає ЧЕРВОНИМ;
  - на весь екран великим текстом виводиться "ТРИВОГА",
    причина, локація і час;
  - сама вкладка Windows Terminal підсвічується червоним
    індикатором, а вікно піднімається на передній план.
Коли оголошують відбій - усе повертається до звичайного
вигляду.

----------------------------------------------------------
  ЯК КОРИСТУВАТИСЯ
----------------------------------------------------------
Відкрий вікно терміналу (cmd, PowerShell або Windows
Terminal) і набери:

    alarm

Працює з будь-якої папки. Зупинка - Ctrl+C (колір
терміналу при цьому автоматично відновлюється).

Альтернатива - запустити монітор в ОКРЕМІЙ вкладці Windows
Terminal (а не в поточній): двічі клацни
run_alert_watcher.bat. Якщо Windows Terminal не
встановлено, відкриється класичне вікно консолі.

----------------------------------------------------------
  ВСТАНОВЛЕННЯ НА НОВОМУ ПК
----------------------------------------------------------
1. Скопіюй усі файли з цієї папки в будь-яку папку на
   новому комп'ютері.
2. Двічі клацни install.bat
3. Відкрий НОВЕ вікно терміналу і набери:  alarm

Інсталятор сам:
  - покладе файли в C:\Users\<ім'я>\bin
  - додасть цю папку в PATH
  - перевірить Python і за потреби встановить бібліотеки
    requests та colorama
  - налаштує АВТОЗАПУСК при старті Windows (фоновий режим)

----------------------------------------------------------
  АВТОЗАПУСК І ФОНОВИЙ РЕЖИМ
----------------------------------------------------------
Після встановлення монітор запускається сам при вході в
Windows: він відкриває окрему вкладку в Windows Terminal,
яка тихо працює у фоні й привертає увагу (червоніє,
піднімається) лише коли оголошують тривогу.

Якщо Windows Terminal недоступний, у фолбек-режимі вікно
тримається ЗГОРНУТИМ і саме спливає на передній план лише
під час тривоги, а після відбою згортається назад.

Файл, що відповідає за фоновий запуск: alarm_background.bat
Ярлик автозапуску "Alarm - Air Raid Monitor" лежить у папці
Startup і створюється інсталятором автоматично.

Щоб ВИМКНУТИ автозапуск - видали цей ярлик з папки
автозавантаження (Win+R -> shell:startup).

ВАЖЛИВО: на ПК має бути встановлений Python 3.
Якщо його немає - встанови з https://python.org
(постав галочку "Add Python to PATH"), потім запусти
install.bat ще раз.

----------------------------------------------------------
  ФАЙЛИ В КОМПЛЕКТІ
----------------------------------------------------------
  alert_watcher.py        - сам скрипт-монітор
  alarm.cmd               - глобальна команда "alarm"
  run_alert_watcher.bat   - запуск в окремій вкладці WT
  alarm_background.bat    - фоновий запуск (для автозапуску)
  install.ps1             - інсталятор (логіка)
  install.bat             - запуск інсталятора (подвійний клік)
  README.txt              - цей файл

----------------------------------------------------------
  НАЛАШТУВАННЯ
----------------------------------------------------------
За замовчуванням стежить за локацією "м. Київ".
Щоб змінити локацію, задай змінну середовища ALERT_LOCATION
перед запуском, напр. у PowerShell:

    $env:ALERT_LOCATION = "Львівська область"; alarm

Назва має точно збігатися з ключем у відповіді API
(області подаються як "<Назва> область", Київ - "м. Київ").

Інтервал опитування - 10 секунд (задано в alert_watcher.py).

----------------------------------------------------------
  ДЖЕРЕЛО ДАНИХ
----------------------------------------------------------
https://ubilling.net.ua/aerialalerts/ - безкоштовний
публічний агрегатор, що проксує офіційні дані (alerts.in.ua,
ukrainealarm.com, канал "Повітряна тривога"). Реєстрація та
API-токен НЕ потрібні.

Примітка: агрегатор повертає лише сам факт тривоги без
підтипу загрози (балістика / БПЛА / крилаті ракети тощо),
тож монітор показує загальну категорію "Повітряна тривога".
